Environmental Pollution Awareness

About This Application

Real-World Problem: Environmental pollution is one of the most critical challenges facing our planet today. It affects air quality, water resources, soil health, and ultimately human health and biodiversity. This application demonstrates various HTML image concepts while raising awareness about different types of pollution.

Purpose of Images: The images used in this application serve multiple purposes:

  • Image Map: The main pollution image contains clickable areas (using rectangle, circle, and polygon shapes) that highlight different pollution sources and their impacts.
  • Background Images: Used to create visual context and enhance the user experience, demonstrating how background images can be applied to entire pages or specific elements.
  • Responsive Images: The <picture> element ensures optimal image display across different screen sizes and devices, improving performance and user experience.

Interactive Pollution Map

Click on different areas of the image to learn about various pollution sources:

Clickable Areas Guide:

  • Circle (Top-right): Vehicle Emissions - Click to learn about transportation pollution
  • Polygon (Center-bottom): Water Pollution - Click to learn about contaminated water sources
  • Rectangle (Bottom-left): Deforestation - Click to learn about forest destruction
  • Circle (Bottom-right): Waste Dumping - Click to learn about improper waste disposal

Take Action Against Pollution

This section demonstrates a background image applied to a specific HTML element. The background image shows a clean environment, representing what we can achieve through collective action.

Background Properties Used:

  • background-size: cover - Ensures the image covers the entire element
  • background-repeat: no-repeat - Prevents image repetition
  • background-position: center - Centers the image
