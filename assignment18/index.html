Hide / Show Demonstration

Box A (will be hidden with display: none)
Box B (will be hidden with visibility: hidden)
Display concepts:
  • block: starts on a new line and takes full width (e.g. headings, div).
  • inline: flows inside a line, width is based on content (e.g. span, a).
  • inline-block: behaves like inline but can have width/height and margin.
  • display: none: element is removed from layout (no space).
  • visibility: hidden: element is invisible but still takes up space.
